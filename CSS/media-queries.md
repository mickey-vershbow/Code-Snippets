
# Tablet Media Query

```
@media screen and (min-width: 480px){
  .container {
    display: grid;
    grid-template-areas:
    "one two"
    "three three"
    "four four";
    background-color: gray;
    width: 400px;
    height: 400px;
    margin: auto;
  }
}
```
